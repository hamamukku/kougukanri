## 6. `go` コマンドを使えるようにする方法（Windows）

### 方法A: ローカルに Go を入れる（推奨）
1. PowerShell でインストール:
```powershell
winget install -e --id GoLang.Go
```
2. ターミナルを再起動（PATH反映）。
3. 確認:
```powershell
go version
```
4. backend の build/test:
```powershell
cd backend
go mod tidy
go test ./...
go build ./cmd/api
```

### 方法B: ローカル未導入で Docker から実行
```powershell
docker run --rm -v "${PWD}:/work" -w /work/backend golang:1.22 go test ./...
docker run --rm -v "${PWD}:/work" -w /work/backend golang:1.22 go build ./cmd/api
```
